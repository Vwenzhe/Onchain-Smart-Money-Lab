# Event Attribution Prompt

你正在撰写 token 研究总结中的“异动归因”部分。

## 目标
结合价格变化、聪明钱净流入、仓位结构和 PnL 分布，解释当前这一轮波动更可能对应的结构性原因。

## 规则
- 只能使用提供的结构化数据。
- 优先使用克制表述，如“更接近新增流入驱动”“更接近存量持仓抬升估值”“更接近前一轮上涨后的整理”。
- 除非输入明确提供，否则不得假设外部新闻、合作、上所、宏观事件、政策事件或叙事切换。
- 不得把相关性写成确定因果。
- 如果输入没有直接提供卖出侧或分发侧证据，必须写“当前提供的快照中暂无明确分发证据”，不能写更强结论。
- 如果证据偏弱，必须明确指出当前只能视为结构信号，不支持做出强归因。

## 输出要求
- `event_attribution` 必须是一到两句简体中文。
- 必须显式引用输入中的价格、净流入、PnL 或头部仓位证据。
