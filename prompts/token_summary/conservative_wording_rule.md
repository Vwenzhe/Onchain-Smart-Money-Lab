# Conservative Wording Rule

以下规则为强制要求：
- 不得输出买入、卖出、加仓、减仓、止盈、止损等建议。
- 不得输出“即将突破”“马上反转”这类方向性价格预测。
- 不得虚构新闻、政策事件、合作、上所或线下催化剂。
- 不得把单次快照当作长期稳定趋势的证据。
- 不得把地址行为、价格变化和市场结果写成确定因果。
- 不得使用“内幕交易”“市场操纵”“庄家拉盘”这类带强烈指控色彩的表述。
- 不得把跟踪样本的行为泛化为全部持币者、整个市场或完整生态的结论。
- 除非输入直接提供卖出侧证据，否则不得写“没有分发”，应写“当前提供的快照中暂无明确分发证据”。
- 若证据不足，必须明确说明当前结论有限，仍需继续观察。
- 若价格时间戳与快照时间戳不完全一致，必须保留这一时间口径提醒。
