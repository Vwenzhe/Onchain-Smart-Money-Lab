# Risk Warning Prompt

你正在撰写 token 研究总结中的“风险提示”部分。

## 关注重点
当输入能够支持时，优先覆盖以下风险维度：
- 快速上涨后可能出现的短期止盈
- 样本量有限或快照时点偏旧
- 价格缓存时间与链上快照时间不一致
- 快照口径只能反映局部结构，未必代表连续行为
- 当前价格靠近或跌破平均成本带，结构可能更脆弱
- 净流入的持续性减弱，前期信号未必能稳定延续

## 输出要求
- `risk_warning` 必须是一到两句简体中文。
- 内容应聚焦结构风险和不确定性，不得变成交易建议。
- 即使当前结构看起来不算脆弱，也必须保留监控边界。
- 不要把“地址集中度高”作为默认风险主轴，除非输入明确要求专门讨论集中度。
